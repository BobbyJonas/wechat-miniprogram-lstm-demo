用 CartoonGAN 及 TensorFlow 2 畫動漫

  • 2019-05-06 (Mon)
  • 1 views

如果能讓 AI 幫我們將圖片自動轉換成新海誠或是宮崎駿等人的動漫風格，豈不是很有意思？

這篇文章展示了一個用 TensorFlow.js 實作的 CartoonGAN，讓你可以直接在瀏覽器裡頭產生動漫風格圖片。

CartoonGAN 於 2018 CVPR 推出，是一個嘗試將真實世界圖片轉換成動漫的對抗生成網路（Generative Adversarial Network, GAN）：

左上為原圖，其餘三圖則為使用 CartoonGAN 以不同動漫風格對原圖轉換後的結果

當初看到這篇論文覺得很有趣，且裡頭剛好展示了兩位我很喜愛的日本動畫作家：新海誠以及宮崎駿的風格轉換結果，因此我決定寫個 TensorFlow.js 應用，讓更多人可以輕鬆地體驗這個有趣的 CartoonGAN。

使用方法很直覺，選擇風格並上傳照片，完成！


動手玩 CartoonGAN¶

你可以在這節試試以 TensorFlow.js 實作的 CartoonGAN。

因為 TensorFlow.js 在瀏覽器裡運行，符合以下條件能讓你獲得較好的體驗：

  • 使用桌電 / 筆電等計算能力較強的機器
  • 在網速快的環境測試（減少載入模型時間）

如果你的設備（如手機）在上傳圖片後一直停在 Loading Model，別走開！下一節提供更強大的方法讓你轉換自己的圖片。

使用 TensorFlow 2 來畫卡通¶

適逢 PoweredByTF 2.0 挑戰，除了 TensorFlow.js 以外我也用了 TensorFlow 2.0 Alpha 來實作 CartoonGAN。

如果你想要轉換更大尺寸的圖片或是動圖，可以參考 CartoonGan-tensorflow 或是馬上點擊這個 Colab 筆記本來轉換圖片：

使用預先準備的 Colab 筆記本可以讓你使用 TensorFlow 2 轉換任何圖片

Google Colaboratory 是 Google 推出的一個雲端 Jupyter 筆記本 環境，並提供 GPU 及 TPU 供任何人開發深度學習相關的專案。

在這個 Colab 筆記本裡頭，以下步驟的程式碼都幫你寫好了：

  • 建置 TensorFlow 2.0 環境
  • 下載 Github Repo 及預訓練模型
  • 下載網路圖片 / gif
  • 使用 CartoonGAN 轉換圖片

你只需打開筆記本並依照指示一步步執行，即可為你的圖片添加動漫風格。

一些轉換後的動漫成果¶

CartoonGAN 的動漫生成當然還有不少改善空間，但已經有不少有趣的結果了。

推薦的 GAN 學習資源¶

近年以深度學習為基礎的生成模型（Generative Models）領域蓬勃發展，其中最亮眼的發展之一當屬對抗生成網路（Generative Adversarial Network, GAN）了。而本文的 CartoonGAN 也是眾多 GANs 裡頭的其中一個小夥子。

對抗生成網路實際上由兩個獨立的神經網路相互「對抗」 （圖片來源）

本文希望專注在 CartoonGAN 的實際應用。對訓練模型、推論的實作細節有興趣的讀者請參考以 TensorFlow 2.0 API 實作的 CartoonGan-tensorflow。

對 GAN 的理論 & 基礎知識有興趣的讀者，我則推薦以下的學習資源：

更多學習資源則請參閱由淺入深的深度學習資源整理。另外如果你有其他推薦的學習資源，還請不吝留言與我及其他讀者分享，謝謝！

結語¶

CartoonGAN 是我第一次嘗試用 TensorFlow 2.0 以及 TensorFlow.js 實作的生成對抗網路，從過程中學到了很多。感謝合作夥伴 mnicnc404 強大的實作支援以及許多寶貴見解，獲益良多。

希望透過這篇文章能讓更多人對強大的 GAN 以及生成模型產生興趣，並進一步探索相關的 AI 知識，嘗試更多的可能性，做出更多有趣的應用：）
