用 CartoonGAN 產生卡通風格圖

  • 2019-05-06 (Mon)
  • 1 views

如果能讓 AI 幫我們將圖片自動加入卡通風格，該有多好？

這篇文章展示了一個用 TensorFlow.js 實作的 CartoonGAN，讓你可以直接在瀏覽器裡頭產生卡通風格圖片。

CartoonGAN 由劉永進教授等人於 2018 CVPR 推出，是一個嘗試將真實世界圖片轉換成卡通的對抗生成網路（GAN）：

左上為原圖，其餘三圖則為使用 CartoonGAN 以不同風格對原圖轉換後的結果

論文裡頭剛好展示了兩位我很喜愛的日本動畫作家：新海誠以及宮崎駿的風格轉換結果，因此我決定寫個 TensorFlow.js 應用，讓更多人可以輕鬆地體驗有趣的 CartoonGAN。

使用方法也很直覺，選擇風格並上傳照片，完成！


CartoonGAN Demo¶

你可以現在就試試我以 TensorFlow.js 實作的 CartoonGAN。不過強烈建議：

  • 使用桌電 / 筆電等計算能力較強的機器
  • 在網速快的環境

下測試以獲得最佳體驗。

如果你的設備（如手機）在上傳圖片以後一直停在 Loading Model，下一節提供別的方法讓你轉換自己的圖片。

使用 TensorFlow 2.0 來轉換圖片¶

除了 TensorFlow.js 以外，我們也使用 TensorFlow 2.0 實作了 CartoonGAN。

如果你想要轉換更大尺寸的圖片或是動圖，可以參考 Github Repo：CartoonGan-tensorflow 或是直接使用這個 Colab 筆記本來轉換圖片：

使用我們預先為你準備的 Colab 筆記本可以讓你轉換任何圖片
