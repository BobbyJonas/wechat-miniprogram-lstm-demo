Generate Anime using CartoonGAN and TensorFlow 2.0

  • Sun, 05 May 2019
  • 1 views

Which image will you choose to transform if you can turn any images into anime using AI?

In this post, I will briefly introduce a CartoonGAN implemented in TensorFlow 2.0 Alpha by my friend mnicnc404 and me.

We will also demo a TensorFlow.js app which allow you to generate anime on the browser directly.

CartoonGAN (original paper) was published in 2018 CVPR and is a Generative Adversarial Network (GAN) which attempt to transform real world images into cartoon-style images.

Left-top is real world image, while all other 3 images are generated by CartoonGAN using different anime styles

The ideas demonstrated in the paper are very interesting and the authors also show good results using two of my favorite Japanese animators' styles: Shinkai Makoto and Miyazaki Hayao.

Therefore, I decided to build a TensorFlow.js app which allow everyone to try CartoonGAN with ease.

Choose your favorite anime style and upload one image, just that simple.


Under the hood, TensorFlow.js run in your browser and it will:

  • download the pretrained models
  • cartoonize uploaded images

To get the best experience, it is highly recommended that you:

  • try the app with desktops/laptops rather than mobile phones (to speed up transformation)
  • try the app when network speed is fast（to minimalize loading time）

In addition to TensorFlow.js, we also implemented CartoonGAN using TensorFlow 2.0 Alpha. If you want to transform larger images and/or gifs, you can run this Colab notebook:

In this notebook, everything you need is set up for you:

  2. Clone our github repo and download pretrained models
  3. Download arbitrary images on the web
  4. Transform the images using CartoonGAN

You got the idea. All you have to do is open the notebook and generate your anime in a breeze.

Using Python script in our github repo, you can generate these results with just one command:

You can imagine apps with similar features will become much more powerful in the near future. Everyone will be able to generate their own anime using various styles, while animators can test their new ideas and draw drafts way more faster than ever before.

Generative models like CartoonGAN can show us many possibilities and inspire us.

Train your own CartoonGAN¶

You may noticed that we've put a lot of focus on actual application rather than the algorithmic details for CartoonGAN. And it’s just because:

  • I believe most of the readers only interested in how to use CartoonGAN rather than how to train a model themselves
  • There are already lots of great learning resources on the web

For those who are interested in implementation details of CartoonGAN, we suggest you taking a look at our github repo. If you have your own datasets, you can even train your own CartoonGAN simply in one command:

Our Python script provide detailed messages which allow you to understand what is happening behind the scenes

Our project can also be used to learn latest version of TensorFlow since everything is implemented in Tensorflow 2.0 alpha:

  • Use tf.keras to implement custom layers and GAN
  • Use tf.data to properly load and process large amount of images
  • Write custom train logic and make computation faster using tf.function
  • Use TensorBoard to monitor model performance in real-time manner

As some of you already knew, it is not trivial to train a GAN. You must keep monitoring your model's performance to decide how to adjust your hyper-parameters accordingly.

In our training script, TensorBoard is integrated perfectly so you can monitor model's performance easily:

One result of many experiments we tried on TensorBoard

In addition to metrics and loss functions, it is good practice to keep an eye on the images generated by GAN as well.

Using our script, checking generated images on TensorBoard is a no-brainer:

the sooner you get feedback, the sooner you can improve your models

Although we only record CartoonGAN-specific metrics and images here, you can easily apply same technique to monitor any other models you like.

Finally, in order to get a sense of how CartoonGAN is doing during training phase, we can save some images and use them as validation set.

The idea here is to "test" CartoonGAN using same images periodically so that we can easily see how it improve over time:

CartoonGAN become better and better transforming validation set into anime

You can see that CartoonGAN learn to transform real world images into cartoon-like images with clear edges and smooth color shading.

Training a GAN like CartoonGAN is far more harder than training a simple classifier. But in our github repo, we try our best to make the TensorFlow code clear and easy to understand, hoping it will help more people to get started.

Needless to say, you can get a better understanding of our code if you know some basics. In the next section, I will list up some good learning resources for your reference.

Recommended learning resources¶

Recently, deep learning-based generative models are very popular and GAN is one of the most exciting research areas. The CartoonGAN we saw in this post is just a simple application in the big GAN zoo.

In a typical GAN setting, there are two independent neural networks, called generator and discriminator, competing with each other （圖片來源）

For those who want to learn more about GAN, I recommend following resources:

  • MIT 6.S191 Deep Generative Models
  • GAN Lab let you train GANs on browser
  • Andrej Karpathy's online GAN demo
  • TensorFlow offical tutorial teach you how to generate MNIST using DCGAN
  • Synced: Reproducing Japanese Anime Styles With CartoonGAN AI
  • Open Questions about Generative Adversarial Networks
MIT 6.S191 Deep Generative Models is a good place to start learning GAN concepts

The researchs on generative models and GAN are evolving rapidly, but I believe you will be able to implement your own GANs after studying these learning resources.

Final words¶

AI technology and applications like CartoonGAN should be more accessable for everyone, not only researchers or machine learning practitioners.

With this in mind, I built this project using TensorFlow 2.0 and hope it can encourge some of you to explore more about GAN and generative models, create more interesting AI applications in the future.

Many thanks to the authors of CartoonGAN, TensorFlow/TensorFlow.js teams and my friend mnicnc404.
